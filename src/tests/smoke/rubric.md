# Evaluation Rubric: smoke

Score the generated code from 1-10 based on the following criteria:

## Correctness (50%)
- Function named `add` is implemented
- Takes two number parameters
- Returns the sum correctly
- Works with positive, negative, and zero values

## Code Quality (30%)
- Clean TypeScript syntax
- Proper type annotations (number parameters, number return)
- Function is exported correctly

## Best Practices (20%)
- Concise implementation
- No unnecessary complexity
- Pure function (no side effects)

## Scoring Guide
- 9-10: Perfect - Correct, clean, properly typed
- 7-8: Good - Works correctly, minor type issues
- 5-6: Acceptable - Basic functionality works
- 3-4: Poor - Partial implementation or errors
- 1-2: Failing - No function or doesn't compile
